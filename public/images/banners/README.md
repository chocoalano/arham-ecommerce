# Banner Images

## Required Images for About Page

Please add the following images to this directory:

1. **about-banner.webp** (800x517px)
   - Main about page banner image
   - Should showcase the company/products

2. **home3-banner1.webp** (350x230px)
   - First promotional banner
   - Links to catalog

3. **home3-banner2.webp** (350x230px)
   - Second promotional banner
   - Links to catalog

4. **home3-banner3.webp** (350x230px)
   - Third promotional banner
   - Links to catalog

5. **home3-banner8.webp** (255x226px)
   - Feature section banner
   - Links to catalog

## Temporary Solution

If you don't have these images yet, you can:
- Use placeholder images from https://placehold.co/
- Copy from your template assets
- Create temporary banners with your logo

## File Format

- Preferred: WebP format for better performance
- Alternative: JPG/PNG format
- Optimize images before uploading
